# DomesticPets

DomesticPets é um site Fullstack feito com Node.js para cuidar da organização do armazém e de informações de Petshops.<br>
<br>

1- Baixar Node.js<br>
2- Instalar as dependências nas pastas frontend e backend ("npm install").<br>
3- Renomear o arquivo .env.exemplo para .env e preencher as variáveis necessárias.<br>
4- O projeto utiliza SQLite como banco de dados (já configurado no backend).<br>
5- Iniciar Aplicação com "npm start", no frontend e backend.<br>
6- Acessar o link preenchido no Navegador.<br>
